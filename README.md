# platzi-tienda-react

This React - Front End project is based on a virtual shop for buying objects. 

## Technologies:
- Frontend: ReactJS, TypeScript, CSS, HTML

## Installation

Execute the command: 

```bash
npm run start
```

## Usage

Go to http://localhost:3000/
